## 1-Phase Dimmer with LCD Display
A PWM pulse synchronized with zero crossing.
 
MCU:        	ATmega32  
Frequency:  	4MHz  
Timer0:		PWM Mode with inverted output  
External INT0:	Rising Mode
	
Note: INT0 used for end to the timer0 task
